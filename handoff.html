Birthday Blow Handoff

Project: birthday-blow-cake | Updated: 2026-06-29

Live URL: https://pornnalin.github.io/birthday-blow-cake/
Debug URL: https://pornnalin.github.io/birthday-blow-cake/?debug=1

Current State

  • Single-page static web app: index.html, styles.css, app.js.
  • Uses MediaPipe Face Landmarker from CDN for face/mouth tracking.
  • Uses the real music file delon_boomkin-happy-birthday-music-box-426283.mp3.
  • Debug overlay is visible by default unless URL has ?nodebug=1.
  • ?debug=1 shows debug and still allows finishing.
  • ?tune=1 keeps debug on but prevents auto-finish for calibration.

Gameplay Flow

  • User taps start, browser asks camera permission, music is unlocked.
  • Cake flies in, screen dims slightly, music starts.
  • Hat tracks the head landmark shown by debug as head.
  • After the ready delay, user blows/puckers mouth long enough to finish.
  • On finish: candle goes out, Happy Birthday to You appears, screen brightens, confetti falls.

Important Code Points

  • Blow timing: BLOW_READY_DELAY, BLOW_HOLD_MS in app.js.
  • Blow detection: isBlowing() in app.js.
  • Hat position: placeHat() in app.js; currently pinned to landmark 10.
  • Cake position: --cake-x and --cake-y in styles.css.
  • Confetti: throwConfetti() in app.js and .confetti-layer CSS.

Known Calibration Notes

  • If the hat still appears away from the head, compare debug values head: and hat:.
  • If it finishes too easily, raise thresholds in isBlowing() or increase BLOW_HOLD_MS.
  • If it never finishes, lower mouthPucker/mouthFunnel from 0.8 slightly.
  • If GitHub Pages looks stale, test with a cache buster such as ?v=3.
